# Expecto VM

## Notes

Interpreters fall in to one of three categories:
  Tree-walking
  Stack-based
  Register-based

## Technical details

opcode (8 bits) - 0, 1, 2, 3 operands
* 0: -
* 1: 24 bits
* 2: 8 bits, 16 bits
* 3: 8 bits, 8 bits, 8 bits
